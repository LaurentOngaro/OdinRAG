# README - templates

## Contents

Internal OdinRAG templates (clone these, do not edit in place):

- `odin-project/` - scaffold for a personal Odin project (AGENTS, README, src/main.odin, devlog template, subagent)
- `planning-daily/` - template for a daily planning entry
- `user_config.example.jsonc` - example `_Private/.config/user_config.jsonc`

## Conventions

These are reference templates. Copy to a new location and edit the copy.
