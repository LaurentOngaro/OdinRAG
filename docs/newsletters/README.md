# docs/newsletters/ - Odin newsletters public index

> This folder is the **public landing** for the Odin newsletters (`odin-lang.org/news/`). Individual issues are **not** redistributed here - see [`../../SOURCES.md`](../../SOURCES.md) § 4.

## How to obtain the issues

- Main index: <https://odin-lang.org/news/>
- Each issue lives at `odin-lang.org/news/newsletter-YYYY-MM/` and is freely readable.

## How to scrape locally

If your scraper targets newsletters (when configured), the output stays in this folder but is **gitignored**. Only this `README.md` is published with the repo.
