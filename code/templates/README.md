# Odin project templates

## Cloning a template

```bash
cd code/templates

# Recommended - hot-reload Raylib
git clone https://github.com/karl-zylinski/odin-raylib-hot-reload-game-template odin-raylib-hot-reload

# Sokol with hot-reload
git clone https://github.com/karl-zylinski/odin-sokol-hot-reload-template odin-sokol-hot-reload

# WASM Raylib build
git clone https://github.com/karl-zylinski/odin-raylib-web odin-raylib-web

# WASM Sokol build
git clone https://github.com/karl-zylinski/odin-sokol-web odin-sokol-web
```

## Catalogue

| Template                       | Main use                                | Cloned |
| ------------------------------ | --------------------------------------- | ------ |
| `odin-raylib-hot-reload/`      | Fast iterative development (recommended) | no     |
| `odin-sokol-hot-reload/`       | Advanced GPU control                    | no     |
| `odin-raylib-web/`             | HTML5 distribution                      | no     |
| `odin-sokol-web/`              | HTML5 distribution + custom shaders     | no     |

> No template is currently versioned in this folder - this is a checklist of projects to clone based on your needs.
