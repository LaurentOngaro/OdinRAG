# README - vendored templates

## Contents

Vendored third-party project templates. **Each subfolder is its own git repository.**

- `odin-raylib-hot-reload/`
- `odin-raylib-web/`
- `odin-sokol-hot-reload/`
- `odin-sokol-web/`

These are **NOT** OdinRAG templates - they are upstream repositories cloned for reference.
The OdinRAG-internal project template lives at [`_Helpers/templates/odin-project/`](../../_Helpers/templates/odin-project/).
