# README - code

## Contents

- `INTEGRATION.md` - convention for placing a personal Odin project under `code/projects/<name>/`
- `examples/` - public Odin example files (e.g. `demo.odin`)
- `gists/` - public Odin gists from awesome-odin
- `templates/` - vendored third-party templates (NOT OdinRAG templates, each is its own repo)
- `projects/` - personal Odin projects (gitignored subprojects)

## Conventions

- Everything in `code/` is public unless a subfolder is gitignored (`code/projects/PVG03_RPG/`).
- For project scaffolding, see `_Helpers/templates/odin-project/`.
